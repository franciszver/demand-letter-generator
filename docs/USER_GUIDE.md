# User Guide - Demand Letter Generator

## Getting Started

### Registration and Login

1. Navigate to the application URL
2. Click "Register" to create a new account
3. Fill in your email, password, and name
4. Select your role (Attorney, Paralegal, or Admin)
5. Click "Register" to create your account

### First Steps

1. **Upload a Document**: Start by uploading source documents (PDF, DOCX, or images)
2. **Generate Letter**: Use AI to generate a draft demand letter from your documents
3. **Refine**: Use AI refinement to improve the letter based on your instructions
4. **Export**: Export the final letter to Word format

## Features

### Document Upload

- Supported formats: PDF, DOCX, JPEG, PNG, GIF, WEBP
- Maximum file size: 50MB
- Drag and drop or click to upload
- Documents are processed automatically in the background

### AI Letter Generation

1. Upload your source documents
2. Wait for processing to complete
3. Optionally select a template
4. Click "Generate Letter"
5. Review the AI-generated draft

### Template Management

Create reusable templates with variables:

- Use `{{variable_name}}` syntax for placeholders
- Common variables: `{{client_name}}`, `{{date}}`, `{{amount}}`
- Templates are applied during letter generation
- Variables are automatically replaced with relevant information

**Example Template:**
```
Dear {{client_name}},

This letter is regarding the matter dated {{date}}.

Sincerely,
{{attorney_name}}
```

### AI Refinement

Refine generated letters with natural language instructions:

- "Make the tone more formal"
- "Add more details about the damages"
- "Emphasize the legal basis"
- "Shorten the letter"

### Real-time Collaboration

- Multiple users can edit the same draft simultaneously
- See who's currently viewing/editing
- Changes are synchronized in real-time
- View change history

### Export to Word

1. Click "Export to Word" button
2. Download link opens in a new tab
3. Download expires after 1 hour
4. Word document includes proper formatting

## Best Practices

1. **Document Quality**: Upload clear, readable documents for best AI results
2. **Templates**: Create firm-specific templates for consistency
3. **Review**: Always review AI-generated content before finalizing
4. **Refinement**: Use specific, clear instructions for best refinement results
5. **Collaboration**: Coordinate with team members when collaborating on drafts

## Troubleshooting

### Document Processing Failed
- Check file format and size
- Ensure document is readable
- Try uploading again

### AI Generation Issues
- Verify document was processed successfully
- Check that document contains relevant legal content
- Try refining with more specific instructions

### Export Issues
- Ensure draft is saved
- Check internet connection
- Try exporting again

## Support

For issues or questions, contact your system administrator.
